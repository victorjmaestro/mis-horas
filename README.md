# Mis Horas

App personal para controlar horario laboral.

## Archivos

- `index.html`: app completa.
- `manifest.json`: configuración PWA.
- `icon.png`: icono de la app.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube estos tres archivos a la raíz del repositorio.
3. Ve a Settings → Pages.
4. En Branch selecciona `main` y carpeta `/root`.
5. Guarda.
6. Abre la URL que GitHub te da.

Antes de usar la versión online, exporta backup desde tu versión local e impórtalo en la versión publicada.
